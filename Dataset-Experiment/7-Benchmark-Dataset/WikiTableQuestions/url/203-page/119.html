This is the list of hangul jamo (letters representing consonants and vowels) including obsolete hangul. This list contains information of location in Unicode mapping.

"Hanyang Private Use" is mapping system which is used in Hangul (word processor). This system maps old Hangul to the Private Use Area in Unicode.

Code points in orange were added in Unicode 5.2.[1]

Code points in pale violet red were corrected in Unicode 5.1 and KS X 1026.
